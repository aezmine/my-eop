Design System & Style Guide

Design principles, variables, and components used across Azmin Hassan's resume-inspired portfolio.

This style guide compiles the design elements that compose the visual language of the portfolio site. Built on a clean, resume-inspired design that balances professional layout with modern developer aesthetics, the system leverages a strict set of design tokens defined as CSS custom properties in style.css.

All styling is implemented using responsive CSS custom variables, fluid grids, and semantic HTML5 elements. Click on any color swatch to automatically copy its CSS custom variable to your clipboard for quick reuse.
